# Flibusta MCP Server 📚

MCP (Model Context Protocol) сервер для поиска, автоматического скачивания, конвертации в PDF и структурированной сортировки книг из библиотеки Флибуста (через OPDS API).

---

## 🚀 Возможности

- **Поиск книг**: Быстрый поиск по названию или автору через OPDS каталог Флибусты.
- **Загрузка**: Прямое скачивание книг в формате `.epub`.
- **Конвертация и сортировка**: Конвертация EPUB в PDF и автоматическое размещение в тематических папках структуры `HUMAN/Человек/<Категория>/`.
- **Приватность**: Все локально скачанные книги и структура библиотеки автоматически внесены в `.gitignore` и не передаются в репозиторий GitHub.

---

## 🛠 Доступные инструменты (MCP Tools)

### 1. `search_books`
Поиск книг на Флибусте по поисковому запросу.
- **Аргументы**:
  - `query` (string, обязательно): название книги или имя автора (например, `"Братья Карамазовы"` или `"Достоевский"`).
- **Возвращает**: Список найденных книг с авторами и прямыми ссылками на скачивание разных форматов.

### 2. `download_book`
Скачивает книгу по прямой ссылке в корень проекта.
- **Аргументы**:
  - `url` (string, обязательно): ссылка на скачивание (из результатов `search_books`).
  - `filename` (string, обязательно): имя файла (например, `brothers_karamazov.epub`).

### 3. `process_and_sort_book`
Конвертирует EPUB в PDF (при помощи Calibre) и перемещает в указанную тематическую категорию.
- **Аргументы**:
  - `filename` (string, обязательно): имя файла в корне проекта.
  - `category` (string, обязательно): названии категории (например: `Вера`, `Духовность`, `Нравственность`, `Мышление`, `Семья`).
  - `convertToPdf` (boolean, опционально, по умолчанию `true`): конвертировать ли в PDF перед перемещением.

## 📦 Установка и сборка

### Предварительные требования
1. **Node.js** (версия 18+)
2. **Calibre** (для консольной утилиты `ebook-convert`, если требуется конвертация в PDF).
3. **Python 3** (для дополнительных конверторов/скриптов).

### Установка зависимостей
```bash
npm install
```

### Сборка проекта
```bash
npm run build
```

---

## ⚙️ Настройка MCP клиента

Добавьте сервер в ваш конфигурационный файл MCP-клиента (например, `claude_desktop_config.json` или конфигурацию Gemini / Roo / Continue):

```json
{
  "mcpServers": {
    "flibusta-books": {
      "command": "node",
      "args": [
        "C:/Users/Admin/Desktop/book-mcp/dist/index.js"
      ]
    }
  }
}
```

## 🐍 Дополнительные Python-скрипты

- `convert_epub_to_pdf.py`: Автономный Python-скрипт для конвертации EPUB в PDF с поддержкой кириллических TrueType шрифтов Windows (Times New Roman / Arial) через ReportLab.
- `scripts/populate_all_books.py`: Скрипт для массового наполнения и структурирования библиотеки.
